# Bookshelf sample app

## build

`$ make debug`

`$ make release`

## api

Install local rest-server with

```
$ yarn global add json-server
```

or

```
$ npm install -g json-server
```

Run server with

```
$ json-server --watch db.json
```

## server

```
$ cd public
$ python3 -m http.server
```

or serve with any other server
